### Installation

To install the application follow the steps below in the console, you need to
have installed nodejs, git and yarn.

Download repo:

```
git clone https://github.com/eddyriv3ra/people-app.git
```

Change to directory:

```
cd people-app
```

Install dependencies:

```
yarn install
```

Run App:

```
yarn start
```
